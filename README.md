# Financial Tracker

A simple C++ application for tracking personal expenses and managing budgets.

## Features

- Set and manage monthly budget
- Add expenses with categories and dates
- View spending statistics by category
- View spending for specific time periods
- Persistent data storage in JSON format

## Building

### Prerequisites
- CMake 3.10 or higher
- C++17 compatible compiler

### Build Instructions

```bash
mkdir build
cd build
cmake ..
cmake --build .
